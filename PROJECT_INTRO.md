# 项目介绍

本项目基于 LangGraph 构建 ReAct 智能体模板，核心目标是提供可扩展、易定制的推理与行动循环，让开发者能快速接入不同模型与工具，完成复杂问题求解。

## 核心设计
- **ReAct 循环**：在 `src/react_agent/graph.py` 中定义，从模型决策到工具执行再回到模型回复，直至满足结束条件。
- **上下文与配置**：`src/react_agent/context.py` 提供系统提示词、模型选择及工具配置，便于通过环境变量或运行时参数调整。
- **状态管理**：`src/react_agent/state.py` 描述对话状态结构，消息按 ID 追加合并，保证对话流转的可追踪性。
- **工具示例**：`src/react_agent/tools.py` 内置 Tavily 搜索示例，开发者可以在此添加或替换为业务相关工具。

## 快速体验
1. 复制环境变量模板：`cp .env.example .env`。
2. 填写所需的模型与 Tavily 搜索 API Key。
3. 在 LangGraph Studio 打开本项目，运行即可与智能体交互。

## 如何扩展
- 在 `tools.py` 中新增 Python 函数，即可作为工具暴露给智能体调用。
- 修改 `prompts.py` 或 Context 中的提示词，使智能体具备特定人格或任务导向。
- 调整 `graph.py` 中的节点与边，改写决策流程或增加额外的控制逻辑。

## 目录速览
- `src/react_agent/`：智能体核心代码（图定义、状态、上下文、工具、提示词等）。
- `static/`：配套的静态资源。
- `tests/`：单元与集成测试目录，可按需补充验证逻辑。
- `README.md`：快速开始与配置指南。

通过以上结构，本项目可作为自定义智能体的起点，既能直接使用默认设置，也能在几处关键文件上进行轻量修改以适配具体场景。富余的扩展点与清晰的注释将帮助你快速完成迭代。
